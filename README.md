## Run
Because morden browsers won't let file directly read models,\
you still need support to run it by using some libraries:\
Run by npm serve:
> npx serve --cors

OR
> npm install serve\
> serve --cors

OR\
Run by python3:
> python3 -m http.server

And access ports relatively
